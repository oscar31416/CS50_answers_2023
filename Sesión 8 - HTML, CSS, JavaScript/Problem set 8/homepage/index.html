What is Lubuntu?

Lubuntu is a distribution of GNU / Linux. It is derived from Ubuntu, a well-known Linux 'distro' which is derived from Debian, another distro that, however, is not so well known.



Unless you want to use nothing but the terminal, a Linux distribution needs a 'desktop environment' as well as a 'window manager'.

Lubuntu includes the LXQt desktop environment and the Openbox window manager for a simple, clean and tidy environment without a high RAM or CPU usage.

Lubuntu includes the basic equipment to start working right after its installation. Among other apps, we can find:

  • A web browser (Mozilla Firefox).
  • A free alternative to Microsoft Office (LibreOffice).
  • An audio and video player (VLC).
  • GUI-based and terminal-based text file editors (FeatherPad, Vim, Nano).
  • A PDF viewer (qpdfview).
  • A Python 3 interpreter, and compilers for C, C++ and Fortran.
  • And lots of terminal-based tools!

At some point, you will need to do things that the starting equipment can't do. Fortunately, the Muon package manager allows you to download (literally) thousands of programmes FOR FREE.

Two types of versions are released by the Lubuntu team:

  • Interim releases.
    Published once every ~6 months.
    As of October 2023, the current one is 23.10, codenamed 'Mantic Minotaur'.

  • Long-term-support releases.
    Published once every 2 years, and supported for ~3.
    The current one, released in April 2022, is 22.04 and its codename is 'Jammy Jellyfish'.
